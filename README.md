## Features

- Pocketbase database deployed on Linode server
- Google Gemini AI chatbot
- NGINX as a reverse proxy for Pocketbase with HTTPS using Let’s Encrypt, with rate limiting
- Icons from Remix icons
- Svelte 5 syntax
